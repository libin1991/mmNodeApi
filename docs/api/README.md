## 分类歌单

获取歌单列表

##### 参数

| 字段 | 默认值 | 可选值 | 是否必传 | 说明 |
| :------------ | :------------ | :------------ | :------------ | :------------ |
| `order` | `163` | `hot` / `new` | `false` | 音乐平台 |
| `limit` | `20` |  | `false` | 返回数量 |
| `offset` | `0` |  | `false` | 偏移数量，用于分页 |

注：
* `order`：`hot` 网易为热门 / QQ为推荐； `new` 最新
* 分页：( 页数 -1)*30, 其中 30 为 limit 的值

##### 请求类型: `get`

##### 接口地址: `/playlist`

##### 调用例子: `/playlist`

![调用例子](/images/playlist.png)

##### 支持平台: 网易云音乐、QQ音乐

##### 更新日期: 2018.07.04

## 搜索

调用此接口 , 传入搜索关键词可以搜索 该音乐 / 专辑 / 歌手 / 歌单 / 用户

##### 参数

| 字段 | 默认值 | 可选值 | 是否必传 | 说明 |
| :------------ | :------------ | :------------ | :------------ | :------------ |
| `keywords` |  |  | `true` | 搜索关键字 |
| `type` | 1 | 1: 单曲 | `false` | 搜索类型 |
| `limit` | 30 |  | `false` | 返回数量 |
| `page` | 0 |  | `false` | 页码 |

##### 接口地址: `/search`

##### 调用例子: `/search?keywords=薛之谦&musicType=163`

![调用例子](/images/search.png)

##### 支持平台: 网易云音乐、QQ音乐

##### 更新日期: 2018.08.03

##### 接口地址: `/search/hot`

##### 调用例子: `/search/hot?musicType=163`

![调用例子](/images/search_hot.png)

##### 支持平台: 网易云音乐、QQ音乐

##### 更新日期: 2018.08.02



## 接口通用参数

| 字段 | 默认值 | 可选值 | 是否必传 | 说明 |
| :------------ | :------------ | :------------ | :------------ | :------------ |
| `musicType` | `163` | `163` / `QQ` | `false` | 音乐平台 |
| `format` | `open` | `open` / `close` | `false` | 接口格式化 |

注：以上参数无论 `post / get` 请求均通过url拼接传入

##### 调用例子: `/banner?musicType=163&format=close`

##### 支持平台: 网易云音乐、QQ音乐

##### 更新日期: 2018.07.16
